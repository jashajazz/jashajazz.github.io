ARIMA-model forecasting¶

ARIMA (AutoRegression Integrated Moving Average) is a classical statistical model that can be used for forecasting time-series.

  • Trend - upward or downward continuous change (increasing sales year by year)
  • Seasonality - seasonal repetitive changes in time serie (high/low seasons of along year calendar)
  • Randomness - random or 'natural' volatility or 'noise' in data (normal changes in sales along normal days)

Time series forecasting can be done utilizing trend and seasonality, but there's no use in random data. Purely random data should have constant variance and is uncorrelated in nature. This can be interpreted with mean and varince metrics.

Components¶

AutoRegressive (AR)¶

AutoRegressive component is calculating the next value based on the regression of previous values in time. AR parameter p defines the time period which is used in forecasting the next value. When p = 1 previous series value is used, p = 2 two previous values are used etc.

Moving Average (MA)¶

Moving Average of random errors in previous values. So to say we calculate the average of the difference between actual and forecasted values of selected period of previous values.

Combining AR and MA we get ARMA which works only for stationary time serie.

Stationary (I)¶

Time serie is stationary if the mean, variance, and coviarnce are constant over time in different time periods. For example increasing trend means that different time period has different means. Regression doesn't work if the time series isn't stationary.

Non-stationary series can be handled or Integrated by differentitating the values. This is done by taking the differences between value as new serie. After taking this into account the non-stationary time series can be stationary.

Autocorrelation¶

When you 'lag' or create a new serie where previous time data point has been moved forward you can see how previous and actual data correlates. So to say how well previous data point correlates with next data point. This can be done with differet amount of lag. Lag 1 is comparing current data point with previous data point taking the correlation. Lag 2 correlates current to data point two time preiods ago etc.

If it is slowely degrading series is not stationary (stasmodel acf graph))

AutoRegressive Integrated Moving Average (ARIMA)¶

Parameters¶

  • AR we give parameter p
  • I we give parameter d
  • MA we give parameter q

Critics¶

ARIMA doesn't take account seasonality which can quite critical for example in sales data. Extending ARIMA model with seasonality is generally known as SARIMA in which the S means ovbiously Seasonality.
