import itertools
import warnings
warnings.filterwarnings('ignore')
p=d=q=range(0,10)
pdq = list(itertools.product(p, d, q))
bestvalue = 99999
bestcombo = ()
for param in pdq:
    try:
        model_arima = ARIMA(train, order=param)
        model_arima_fit = model_arima.fit()
        print(param, model_arima_fit.aic)
        if model_arima_fit.aic < bestvalue:
            bestvalue = model_arima_fit.aic
            bestcombo = param
    except:
        continue
print('Best combo:', bestcombo, '\n', 'Best value: ', bestvalue)

Now we have our predictions for starting companies for next four quarters. There seems to be slight shits on the time series in the last couple of years which might make it more difficult to predict. Predictions are created using historic data which should be understood as looking at the numbers.

There seems to be certain pattern that highest number of new company registrations are made in the first quarter and next three quarter decreases. This might have to do with the availability of the funding for startups or simply timing of the registration. Keeping that in mind there might be interesting other independent variables that could affect the amount also.
